> **Programación Web Avanzada**

## **Descripción del Repositorio**
Este repositorio contiene todos los ejemplos de código, proyectos, ejercicios y materiales utilizados durante la asignatura **Programación Web Avanzada**.  
Su propósito es servir como guía de referencia para los estudiantes, documentar los avances desarrollados en clase y proporcionar una base sólida para futuros proyectos web.

---

## **Objetivos del Curso**
- Comprender conceptos avanzados del desarrollo web moderno.
- Utilizar frameworks, librerías y herramientas actuales del ecosistema JavaScript.
- Construir APIs robustas, sistemas backend y aplicaciones frontend interactivas.
- Implementar principios de arquitectura, seguridad, pruebas y despliegue.
- Integrar servicios en la nube, bases de datos y flujos CI/CD.
- Desarrollar pensamiento crítico y habilidades de diseño de software.

---

## **Tecnologías Principales**

### **Frontend**
- HTML5, CSS3, JavaScript ES6+
- Frameworks: Angular, React, Vue.js
- Integración con APIs REST
- Componentes reusables, hooks, stores, composición

### **Backend**
- Node.js
- Express.js
- Controladores, middlewares, patrones de diseño
- Autenticación con JWT, gestión de sesiones
- Validación de datos con herramientas modernas

### **Bases de Datos**
- PostgreSQL, MySQL o MongoDB
- ORMs/ODMs como Prisma, Sequelize o Mongoose
- Consultas avanzadas y modelado de datos

### **Herramientas Avanzadas**
- Git y GitHub
- Postman, Hoppscotch
- Docker
- Firebase, NetlifY, Vercel, Railway
- Variables de entorno y buenas prácticas de seguridad
